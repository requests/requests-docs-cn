更新
=======

如果你想和社区以及开发版的 Requests 保持最新的联系，
这有几种方式:

GitHub
------

最好的方式是追踪 Requests 开发版本的
`GitHub 库 <https://github.com/kennethreitz/requests>`_.

Twitter
-------

我经常推送关于 Requests 的新功能和新版本.

关注 `@kennethreitz <https://twitter.com/kennethreitz>`_ 即可获得更新。
